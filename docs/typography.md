# Typography Direction

## Overview

Typography balances **friendly brand expression** with **high functional readability**.
Decorative typography is intentionally isolated from functional UI elements.

---

## Font Selection

### Decorative Headline Font (Default)

**Pacifico**

**Optional Alternative**
**Sacramento**

**Purpose**

- Brand personality
- Friendly, welcoming tone

**Usage**

- Page-level headlines (e.g. Login, Welcome Back)
- Single-line decorative titles only

**Rules**

- Never used for functional UI
- Never used for buttons, labels, or inputs
- Not used for multi-line text
- No hierarchy depends on font weight
- Font can be swapped without layout changes

---

### Functional / Content Font

**Plus Jakarta Sans**

**Purpose**

- Readability
- Structural hierarchy
- Accessibility

**Usage**

- Section headings
- Body text
- Form labels
- Inputs
- Buttons
- Helper and error text

**Rules**

- All functional hierarchy is handled by this font
- Weight variations define importance
- Must remain readable at small sizes

---

## Hierarchy Strategy

- Font-family switch separates emotional and functional roles
- Weight and size are applied only to Plus Jakarta Sans
- Decorative font never participates in layout structure

---

## Accessibility Considerations

- Script fonts are excluded from critical UI
- All form-related text uses a readable sans-serif font
- Text contrast and size follow accessibility best practices

---

## Summary

- Pacifico provides warmth and clarity by default
- Sacramento remains an optional stylistic alternative
- Plus Jakarta Sans ensures usability and scalability
